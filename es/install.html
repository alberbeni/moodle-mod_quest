Instalar

Cómo instalar QUESTOURnament en su servidor Moodle.

Opción 1: Instalar desde Moodle

Puede instalarlo directamente desde el directorio de complementos de Moodle a través del menú Administración del sitio> Complementos> Instalar complementos. Símplemente navega por el directorio y selecciona instalar. (Tu servidor Moodle necesita estar registrado).

Opción 2: descarga el módulo

Puede descargar el módulo usando el botón "Download" en la  página de QUESTOURnament del directorio de plugins de Moodle. Tras descargar el paquete ZIP, descomprímelo en la carpeta /mod del directorio de tu Moodle.

Opción 3:  descargar las fuentes desde Github

Questournament es Open Source por lo que puede obtener el código fuente del repositorio GitHub.

Instalando y conigurando

Inicie sesión en su página web de Moodle como administrador y visite la página de administración. Aparecerán todos los complementos que se actualizarán o instalarán, incluido el módulo QUESTOURnament. Haga clic en el botón Actualizar base de datos de Moodle para continuar con la instalación. Siga las instrucciones en pantalla.
