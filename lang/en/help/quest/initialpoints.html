Initial Score

This parameter sets the initial score of the challenge to be added.

However, the teacher will be able to modify this value till the maximum score set for the QUESTOURnament where the challenge is to be added. A student can also modify this initial score when adding a challenge, but only by choosing a value equal or lower than the value set by the teacher as initial score of the QUESTOURnament where the challenge is to be added.

Finally, it is important to emphasize that the value of this parameter can be different for each challenge in a QUESTOURnament.
