Ventana de Previsualización del Desafío

Esta página muestra el aspecto que tiene el desafío que se está editando en el editor de Moodle después de aplicarse los distintos filtros que proporciona dicha plataforma.

De esta manera se podrá comprobar el aspecto de un desafío antes de añadirlo. Esta posibilidad resulta sobre todo útil cuando se necesite introducir una ecuación matemática para lo cuál se tendrá que utilizar una expresión con el formato adecuado que transformarán los filtros que proporciona Moodle y cuyo aspecto final no puede visualizarse en el editor de dicha plataforma.
