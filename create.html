Create an instance

QUESTOURnament is an activity that is included on the main page of the course and generates a grade in the grade book.

Activate the editing mode

Choose the topic or section of the course where you want to include the activity.
Select the activity QUESTOURnament. You will find it in the "ACTIVITIES" section.

Fill in the title, description and other standard fields

QUESTOURnament has the standard fields usual in many other activities. With the default settings you already have a functional QUESTOURnament.
Enter at least the title of the competition and the description with its multimedia elements.

Setting up the evaluation guide

As the students will be evaluators of the answers it is convenient to make the evaluation simpler by providing as concrete a guide as possible. QUESTOURnament will show you the editor of the evaluation guide if it is not defined.

Create valuation elements

Each element must cover a particular aspect to be evaluated in the responses sent and its number will depend on the size and complexity of the challenge proposed. The elements will have the following characteristics:
- DESCRIPTION of the evaluation element. It should clearly state what aspect of the task is being evaluated. If the evaluation is qualitative, it will be useful to give details of what is considered excellent, average, etc.
- The SCALE of the evaluation element. There are a number of predefined scales, ranging from a simple YES/NO scale to a full percentage scale. Each assessment element has its own scale, which should be chosen so that it can be adjusted to all possible variations of that element. However, it should be noted that the scale does NOT determine the importance of the element when calculating the final rating: for example, an element with a two-point scale has the same "influence" on the final rating as another element using a 100-point scale, provided that the respective elements have the same weight or weighting factor.
- The WEIGHT of the evaluation element. By default, all elements have the same importance when calculating the final grade of the task. However, this fact can be altered by giving the most important elements a weight greater than one, and the least important elements a weight less than one. It should be noted that modifying the weight does NOT affect the possible maximum rating of an answer, since that value is determined by the Maximum Score set for the challenge for which the answer is sent and for the instant in which the answer is sent. 

Each "challenge" uses its own evaluation guide.

Basic Parameters

Settings common to other modules

As with many other Moodle activities, you have to enter the title, description, file attachments, and the start and end dates of the competition.

QUESTournament settings

Validate Evaluations: When students evaluate responses to their own challenges it is possible to retain them until a teacher validates them.
Use Password: Allows you to control which students access the contest by providing a password.
Allow Teams: Activates the team mode. In this mode students must form teams and achievements are shared.
Number of Components: Maximum team size.

Specific parameters

Configuring Contest Behavior

Control of anonymity

Show Individual Classification: Allows both the individual classification of students and the classification by teams to be shown if the option: Allow Teams is activated.

Show students details of Author in ranking: If this option is enabled, the scoring table will show detailed data on the number of challenges proposed and assessed by each student and by each team, in addition to the score obtained by the responses sent to proposed challenges. It should be noted that this prevents the anonymity of the authors of the challenges from being respected, especially in small groups. 

Show Students the Authors of the Challenges once closed: If this option is enabled, the names of the authors of the challenges of a QUESTOURnament module will be shown when those challenges have been closed and it is no longer possible to send more answers.

Evaluation and Grading

Characteristics of the grading: The options in this section define what the evaluation guidelines are for evaluating the quality of the answers and for evaluating the quality of the questions.
Grading characteristics: The options in this section regulate how the timing and quality of responses are transformed into student and team scores.
